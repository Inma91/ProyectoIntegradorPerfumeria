Sprint 3 — Junio 2026
Autor: Brandon Gaviria
Equipo: Brandon Gaviria · Inmaculada Gil · David Moreno


**1. INTRODUCCIÓN**


Este documento recoge las pruebas de integración realizadas
sobre las vistas de la aplicación Golden Tale, desarrollada
en Java Swing con arquitectura MVC. El objetivo es verificar
que cada vista interactúa correctamente con el controlador
y produce los resultados esperados ante las acciones del usuario.

Las pruebas de integración complementan las pruebas unitarias
realizadas por Inmaculada Gil con JUnit, y cubren los flujos
completos: entrada de datos en la vista → procesamiento en el
controlador → actualización de la vista y/o la base de datos.


**2. ENTORNO DE PRUEBAS**


Lenguaje:            Java 17
Framework UI:        Java Swing
Base de datos:       SQLite 3
IDE:                 Eclipse IDE
Sistema operativo:   Windows 10/11
Arquitectura:        MVC — Controlador único (ActionListener)
Datos de prueba:     BD con 15 perfumes, 20 usuarios, 15 pedidos
Fecha ejecución:     Junio 2026
